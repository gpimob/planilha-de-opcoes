# GP IMOB – Lista de Imóveis

Projeto web para listagem e filtragem de imóveis de médio e alto padrão.

## Funcionalidades
- Filtro por nome, bairro, valor e tipologia
- Ordenação por valor
- Limpar filtros
- Layout premium (GP IMOB)
- Responsivo
- GitHub Pages

## Tecnologias
- HTML5
- CSS3
- JavaScript Vanilla
